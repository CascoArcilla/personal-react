## Rol
Como desarrollador y diseñador frontend

## Contexto
Quiero crear mi pagina web personal con vite, react y tailwindcss, donde plasmare mi persona, mis conocimientos (como los que estoy aprendiendo en estos momentos), mis proyectos (creados, en los que estoy trabajando)

## Tareas
Necesito que inicies el proyecto con vite, react y tailwincss, crees la pagina y/o plantilla con las secciones bien definidas componetizada, modular para agregar mas secciones en el futuro.

## Restricciones
El diseño debe ser responsivo, animaciones ligeras y suaves (usar una libreria), tener tanto modo claro como modo oscuro (definir bien la paleta de colores para posteriores cambios en el futuro), para imagenes que se lleguen a mostrar ocupar un lazy loading, incluir un nav (en pantallas chicas para celulares, debe tener un diseño en el que se mustre debajo y cerca del dedo, para pantallas grandes esta bien mostrarlo arriba), no usar formularios para contacto, solo es para presentar informacion, usar el manjador de paquetes por pnpm.

## Formato
Crear la estructura de carpetas adecuada para cada sección de la app.
