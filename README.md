# Norberto Montalvo — Portfolio

Mi página personal, construida con React, Vite y Tailwind CSS. Incluye modo oscuro, animaciones con Framer Motion y despliegue automatizado a GitHub Pages.

## Stack

- **Framework:** React 19 + JSX
- **Build:** Vite 8
- **CSS:** Tailwind CSS v4
- **Animaciones:** Framer Motion 12
- **Iconos:** react-icons
- **Linter:** oxlint
- **Package Manager:** pnpm

## Secciones

- **Inicio** — Presentación principal
- **Sobre mí** — Biografía y foto
- **Skills** — Tecnologías y herramientas
- **Proyectos** — Proyectos realizados
- **Experiencia** — Trayectoria profesional y académica
- **Contacto** — Redes y email

## Scripts

```bash
pnpm dev
pnpm build
pnpm preview
pnpm lint
```

## Despliegue

El proyecto se despliega automáticamente a GitHub Pages mediante GitHub Actions al hacer push a `main`. El workflow construye el sitio y lo publica en la rama `gh-pages`.
